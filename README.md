# ASD VAE Project

## Descripción
Proyecto de Variational Autoencoder (VAE) para análisis de datos ASD.

## Instalación
```bash
pip install -r requirements.txt
```

## Uso
1. Prepara tu conjunto de datos en la carpeta `data/`
2. Ejecuta el entrenamiento:
    ```bash
    python train.py
    ```
3. Ejecuta el test
     ```bash
    python test.py
    ```
4. Genera predicciones:
    ```bash
    python predict.py
    ```

## Estructura
- `train.py` - Script de entrenamiento
- `predict.py` - Script de predicción
- `../data/` - Carpeta de datos
- `model_output/` - Modelos guardados

## Requisitos
- Python 3.8+ COMPROBAR
- TensorFlow/PyTorch
- NumPy, Pandas
